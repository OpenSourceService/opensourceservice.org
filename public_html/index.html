Open-Source Services

This is the home page to document the exploration of a concept I am calling an "open-source service."

What's the big idea here?

Definition

An open-source service is an Internet application developed, managed, and operated transparently by a public, collaborative community.

The Question

Can a community manage a web service or application while operating in complete transparency?

Transparency

As with open-source software, an open-source service develops the sofware public view, dislosing all code, changes, issues, plans, etc.

A completely transparent service will provide access to server configurations, logs, & analytics while protecting user accounts, data, and usage.

Goals

  • Take what is great about open-source software and give it heartbeat
  • Develop best practices for community-operated public Internet services
  • Discover how transparency affects security and the experience of users and operators
